# 📦 Retail Sales Forecasting & Inventory Optimization System

Forecast demand. Optimize inventory. Reduce stockouts.

---

## 🚀 Project Overview
This project is an end-to-end Retail Analytics system that predicts product demand and generates intelligent inventory decisions such as Safety Stock, Reorder Point, and Order Quantity.

It simulates real-world retail operations and demonstrates how businesses optimize stock levels to reduce losses and improve efficiency.

---

## 🎯 Problem Statement
Retail businesses often face:
- Stockouts → lost sales  
- Overstock → high holding costs  

This project solves these problems using data-driven forecasting and inventory optimization techniques.

---

## 💼 Industry Relevance
Used in companies like:
- Amazon  
- Walmart  
- Flipkart  
- Reliance Retail  

These systems help improve:
- Demand planning  
- Supply chain efficiency  
- Profitability  

---

## 🧠 Features
- Time-series sales forecasting  
- Feature engineering (lags, rolling averages)  
- Machine Learning model (Random Forest)  
- Inventory optimization:
  - Safety Stock  
  - Reorder Point  
  - Order Quantity  
- Interactive dashboard using Streamlit
